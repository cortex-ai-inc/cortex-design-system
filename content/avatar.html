Avatar

User avatar display with image support and fallback initials.

React Radix Avatar

Overview

The Avatar component uses Radix Avatar for accessible image loading with fallback support. Avatars are rounded-sm (4px square) with h-10 w-10 (40x40px) default size. When an image is provided, it loads in the AvatarImage component. If the image fails to load or is not provided, the AvatarFallback displays initials with a deterministic background color.

Utilities

Two utility functions provide consistent avatar generation:

getInitials(name: string): string

Extracts initials from a full name. Returns the first character of the first and last name segments, uppercased.

  • getInitials("John Doe") → "JD"
  • getInitials("Alice") → "A"
  • getInitials("") → "?"
  • getInitials("John Michael Doe") → "JD"
getAvatarColor(name: string): string

Returns a deterministic HSL color string based on the name hash. Used as the fallback background color.

  • Maps name to a hue (0-360)
  • Saturation: 45%
  • Lightness: 35%
  • Consistent across renders

Usage Guidelines

Do

  • Always provide alt text on AvatarImage
  • Use getInitials() + getAvatarColor() for fallbacks
  • Use stacked avatars to show team membership
  • Keep avatar sizes consistent within a context

Don't

  • Use avatars without fallback — images can fail to load
  • Mix different avatar sizes in the same group
  • Place more than 5-6 avatars in a stack before using +N overflow
