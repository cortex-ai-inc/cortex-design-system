Shadows

Minimal shadow usage — prefer tonal layering over drop shadows.

Shadow Tokens

TokenValueUsage
shadow-xs 0 1px 2px 0 rgb(0 0 0 / 0.05) Subtle elevation for small floating elements
shadow-ambient 0 32px 64px rgba(0, 0, 0, 0.12) Modal and dialog ambient occlusion

When to Use Shadows

Shadows are reserved exclusively for floating elements: modals, dropdowns, tooltips, and popovers. Cards and panels use tonal layering instead.

💡 Design Principle

Surface colors create depth: surface-dim → surface-container-lowest → surface-container-low → surface-container-high. Each level sits "higher" than the last without requiring a drop shadow.
