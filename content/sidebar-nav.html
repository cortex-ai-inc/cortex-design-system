Sidebar Navigation

Primary navigation rail for authenticated application layouts.

React Radix Dialog (mobile)

Overview

The sidebar provides persistent navigation for authenticated application views. On desktop, it is a fixed element at w-[260px] that can collapse to w-[72px] (icon-only mode). On mobile, it renders as an overlay panel with a backdrop, triggered by a hamburger button in the topbar.

The sidebar background matches bg-surface-dim (#111319) with a right border of border-r ghost-border. The topbar sits alongside it in the remaining viewport space.

Usage Guidelines

Do

  • Use icons consistently — either all outlined or all filled
  • Group related routes under section headers
  • Show active state with both background change and left border indicator
  • Collapse to icon-only on smaller desktop screens (below 1024px)

Don't

  • Put more than 2 levels of nesting in the sidebar
  • Mix navigation with actions — use bottom section for secondary actions
  • Hide the sidebar completely on desktop — use collapse instead
