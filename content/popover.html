Popover

Floating contextual panel anchored to a trigger. Foundation primitive for Combobox and DatePicker.

New React Radix Popover

Overview

Popover renders free-form panel content anchored to a trigger element, built on Radix Popover. The panel sits on bg-surface-container-low with a hairline border-on-surface-variant/10 border, rounded-md (6px) corners, shadow-overlay elevation, and a 200ms animate-fade-in entrance. Default size is w-72 p-4; composed content (Combobox lists, DatePicker calendars) overrides with w-auto p-0.

Popovers are non-modal — the page stays interactive, Esc closes and returns focus to the trigger. Use DropdownMenu for action menus; Popover is for panels.

Usage Guidelines

Do

  • Use for non-modal contextual panels: filters, quick settings, inline pickers
  • Override to w-auto p-0 when composing Combobox or DatePicker content
  • Match panel width to the trigger with w-[--radix-popover-trigger-width]

Don't

  • Use for action menus — use DropdownMenu instead
  • Put create/edit forms inside — forms get dedicated pages
  • Use for confirmations — use AlertDialog
