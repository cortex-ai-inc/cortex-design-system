With Actions

Item with actions

The Item component follows a compositional pattern with ItemGroup for grouping related items, ItemMedia for avatars/icons, ItemContent for main content, ItemActions for action buttons, and ItemHeader / ItemFooter for header/footer sections. Variants: default (bg-surface-container-low), outline (transparent with border), muted (bg-muted/50).

API

Prop Type Default Description
variant "default" | "outline" | "muted" "default" Visual variant of the item.
size "default" | "sm" "default" Item padding. default=p-4, sm=py-3 px-4.
className string "" Additional CSS classes.

Usage

{`import { Item, ItemMedia, ItemContent, ItemActions, ItemTitle, ItemDescription, ItemHeader, ItemFooter } from '@/components/ui/item'

// Basic item
<Item variant="default">
  <ItemMedia>
    <Avatar>JD</Avatar>
  </ItemMedia>
  <ItemContent>
    <ItemHeader>
      <ItemTitle>John Doe</ItemTitle>
      <span className="text-muted text-body-sm">2h ago</span>
    </ItemHeader>
    <ItemDescription>Message content here</ItemDescription>
  </ItemContent>
  <ItemActions>
    <Button variant="ghost" size="sm">Reply</Button>
  </ItemActions>
</Item>

// With ItemGroup
<ItemGroup>
  <Item>...</Item>
  <Item variant="outline">...</Item>
  <Item variant="muted" size="sm">...</Item>
</ItemGroup>`}
