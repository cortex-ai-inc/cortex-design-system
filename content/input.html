Input & Textarea

Text input fields for single-line and multi-line data entry.

React

Overview

Input fields follow a consistent styling pattern: h-10 (40px) height, bg-surface-dim fill, and a ghost-border-20 border. On focus, inputs display a ring-2 ring-k-primary ring. All inputs use rounded-sm (4px) corners and px-3 py-2 padding. Placeholder text uses on-surface-variant (muted) color.

Textarea

Textareas use the same visual styling as inputs but have a min-h-[80px] minimum height and allow vertical resizing. They are used for multi-line content like descriptions and comments.

Input with Icon

When an input is accompanied by an icon (e.g., search), the icon sits inside the input area to the left, with the input text offset appropriately. The icon uses text-on-surface-variant color.

Usage Guidelines

Do

  • Always pair inputs with labels
  • Use appropriate type for email, password, etc.
  • Show error messages inline below the field
  • Use Input with icon for search fields

Don't

  • Use placeholder text as a label substitute
  • Remove focus rings for custom styling
  • Use inputs without proper aria-label or associated label
