NotificationBell

A notification bell component with unread count badge and dropdown panel. Displays real-time notification updates with type-specific icons.

React Radix

States

3
Unread Count
Empty (No Unread)
Loading

Dropdown Panel

Notifications
New comment on TKT-0042
John added a comment to your ticket.
5 minutes ago
Status changed: TKT-0040
Ticket moved to Resolved.
1 hour ago
Assigned to TKT-0039
You have been assigned to a new ticket.
3 hours ago

Notification Types

Type Icon Color Description
comment MessageSquare Blue (#ADC6FF) New comment on a ticket
status_change ArrowRight Amber (#FBBF24) Ticket status has changed
assigned UserPlus Green (#4ADE80) User assigned to a ticket

Notifications poll every 30 seconds via React Query refetchInterval: 30000. The last-check timestamp is stored in localStorage as cortex.notifications.since. Unread count badge shows on the Bell/BellDot icon. Notifications are auto-marked as read when the dropdown opens.

API

Prop Type Default Description
className string -- Additional CSS classes for the trigger button

{`import { NotificationBell } from '@/components/ui/notification-bell'
