DatePicker

Single dates and ranges picked from a react-day-picker calendar inside a Popover, with an Input-height trigger.

New React Radix Popover react-day-picker

Overview

The DatePicker is a composition recipe: a react-day-picker Calendar inside a Radix Popover, opened from a Button variant="outline" restyled to the Input field tokens (h-9, ghost-border-20, bg-surface-dim, rounded-sm, left-aligned). Values are formatted with date-fns. This page doubles as the Calendar documentation — day cells are rounded-sm squares (never circles), selected days use bg-k-primary-container with dark text-surface-container-lowest, and today is outlined, not filled. Range mode tints the middle days with bg-k-primary-container/20 and uses numberOfMonths={2} in products.

Usage Guidelines

Do

  • Keep the trigger h-9 so it aligns with sibling Inputs and Selects
  • Format values with date-fns — "PPP" single, "LLL dd, y" range ends
  • Use mode="range" with numberOfMonths={2} for reporting filters
  • Add the presets rail when users repeat the same relative ranges
  • Disable un-selectable dates with the disabled matcher

Don't

  • Page decades back for far-past dates — use a dropdown caption (month/year selects)
  • Render a DatePicker inside a modal — forms live on dedicated pages
  • Make day cells circular — rounded-sm squares only
  • Fill today blue — today is outlined; the blue fill means selected
  • Put light text on selected cells — dark text-surface-container-lowest on k-primary-container
