Collapsed Sidebar

When collapsed, only icons are shown. The toggle button in the topbar switches between expanded and collapsed states. On mobile, the sidebar is hidden by default and opens as an overlay.

Breadcrumb Route Mapping

Breadcrumbs derive labels from a routeNames object that maps route paths to human-readable labels.

Route Name Mapping 
const routeNames: Record<string, string> = {
  '/':                    'Home',
  '/dashboard':           'Dashboard',
  '/tickets':             'Tickets',
  '/tickets/new':         'New Ticket',
  '/tickets/:id':         'Ticket Detail',
  '/profile':             'Profile',
  '/admin':               'Administration',
  '/admin/tickets':       'All Tickets',
  '/admin/tickets/:id':   'Ticket Detail',
  '/admin/team':          'Team Management',
  '/admin/settings':      'Settings',
};

Mobile Hamburger Menu

On viewports below the md breakpoint, the sidebar is hidden and replaced by a hamburger button in the topbar. Tapping it opens the sidebar as a full-height overlay with a semi-transparent backdrop.

Do: Use a semi-transparent backdrop (bg-black/50) behind the overlay sidebar on mobile.
Don't: Push the content to the right on mobile — always use overlay.
