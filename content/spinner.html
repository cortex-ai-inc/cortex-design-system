Spinner

A simple spinning border indicator for loading states. Supports multiple sizes and can be used inline or as an overlay.

React

Sizes

sm (w-4)
md (w-5)
lg (w-8)

Usage Contexts

Loading tickets...
Saving...

The Spinner uses a CSS border animation technique. It consists of a circular border with border-k-primary-container/30 on three sides and border-t-k-primary-container on the top, rotating via animate-spin.

API

Prop Type Default Description
size "sm" | "md" | "lg" "md" Spinner size. sm=16px, md=20px, lg=32px.
className string "" Additional CSS classes.

Usage

{`import { Spinner } from '@/components/ui/spinner'

// Basic usage
<Spinner />

// With size
<Spinner size="sm" />
<Spinner size="lg" />

// With label
<div className="flex items-center gap-3">
  <Spinner />
  <span>Loading...</span>
</div>`}
