Select

Dropdown selector for choosing from a predefined list of options.

React Radix Select

Overview

The Select component uses Radix UI Select for accessible, keyboard-navigable option picking. The trigger button matches the Input styling (h-10, bg-surface-dim, ghost-border-20, rounded-sm) for visual consistency in forms. The dropdown content appears at z-50 with shadow-ambient elevation.

Structure

Component Role
Select Root context provider for value + open state
SelectTrigger Visible trigger button (styled like Input)
SelectValue Displays the current selected value
SelectContent The dropdown panel (z-50, shadow-ambient)
SelectItem Individual option in the list
SelectLabel Section label within the dropdown
SelectSeparator Divider between option groups

Usage Guidelines

Do

  • Use Select for 5+ options — prefer RadioGroup or ToggleGroup for fewer
  • Provide a meaningful default or placeholder option
  • Group related options with SelectLabel and SelectSeparator

Don't

  • Use Select for navigation — it's for value selection
  • Make options too long — keep text concise
  • Use placeholder as the only label — always pair with a visible label
