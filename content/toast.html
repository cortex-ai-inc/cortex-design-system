Toast / Sonner

A toast notification system powered by Sonner. Provides lightweight, customizable notifications for success, error, and informational messages.

React sonner

Toast Types

Icon Mapping

Type Icon Color Usage
success CheckCircle Green (#4ADE80) Operation completed successfully
error XCircle Red (#F87171) Error or failure occurred
info Info Blue (#ADC6FF) Informational message
warning AlertTriangle Yellow (#FBBF24) Warning or caution
loading Loader Blue (#4D8EFF) Loading / in progress

Toasts use the Sonner library with a dark theme. Import the Toaster component once at the root layout level and call toast.success('message') from anywhere in your app. The toaster renders in a fixed position at the bottom-right of the viewport by default.

API

Method Signature Description
toast.success (message: string) => string Show a success toast
toast.error (message: string) => string Show an error toast
toast.info (message: string) => string Show an info toast
toast.warning (message: string) => string Show a warning toast
toast.loading (message: string) => string Show a loading toast (returns toast ID for dismissal)
toast.dismiss (id?: string) => void Dismiss a toast by ID, or all toasts

Usage

{`import { Toaster } from 'sonner'

// In root layout:
<Toaster
  position="bottom-right"
  theme="dark"
  richColors
  closeButton
/>

// Anywhere in app:
import { toast } from 'sonner'

toast.success('Ticket created successfully')
toast.error('Failed to create ticket')
toast.info('Your request is being processed')`}
