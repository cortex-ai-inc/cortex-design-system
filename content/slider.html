Slider

Pick a value — or a range — from a bounded interval by dragging a thumb along a track.

New React Radix Slider

Overview

The Slider is built on @radix-ui/react-slider. Its track is the Progress track verbatim (h-2 rounded-full bg-surface-container-high) and its thumb mirrors the Switch thumb (h-4 w-4 rounded-full bg-on-surface), so the three controls read as one family. The filled range uses bg-k-primary-container. Values are arrays: [50] renders one thumb, [20, 60] renders a two-thumb range. rounded-full here is the same exemption Progress and Switch use — track, range, and thumb are inherently circular/pill elements.

Usage Guidelines

Do

  • Use for bounded numeric ranges — volume, thresholds, confidence cutoffs
  • Pair with a value label row so the current value stays readable
  • Use two thumbs + minStepsBetweenThumbs for min/max filters
  • Use onValueCommit for expensive side effects, onValueChange for live labels
  • Keep the 2px focus ring — arrow keys move the thumb by step

Don't

  • Use for precise numeric entry — use Input type="number"
  • Use for 2–5 discrete options — use a RadioGroup
  • Restyle track or thumb to rounded-sm — same rounded-full exemption as Progress/Switch
  • Change the geometry — track h-2 and thumb h-4 w-4 are fixed
  • Color the thumb blue — the thumb is light; blue is the range fill
