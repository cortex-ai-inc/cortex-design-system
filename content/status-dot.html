StatusDot

A small colored dot indicator representing status or state. Supports multiple status colors, sizes, and an optional pulse animation for active states.

React

Status Examples

Active / Success
Inactive / Offline
Failed / Error
Running
Warning
Pending

Sizes

sm (6px)
md (8px)

Color Map

Status Color CSS Variable Pulse
active / success Green (#4ADE80) --k-secondary Yes
inactive / offline Gray (#C2C6D6) --on-surface-variant No
failed / error Red (#F87171) --k-error No
running Blue (#ADC6FF) --k-primary Yes
warning Yellow (#FBBF24) --k-warning No
pending Gray (#C2C6D6) --on-surface-variant No

The pulse-ring animation is applied to active/success and running statuses to draw attention. The dot renders as a span with rounded-full and the appropriate background color.

API

Prop Type Default Description
status StatusDotStatus "inactive" Status key for color mapping.
size "sm" | "md" "md" Dot size. sm=6px, md=8px.
pulse boolean false Show pulse ring animation.
className string "" Additional CSS classes.

Usage

{`import { StatusDot } from '@/components/ui/status-dot'

// Basic
<StatusDot status="active" />

// With pulse
<StatusDot status="running" pulse />

// Small size
<StatusDot status="success" size="sm" />

// With label
<div className="flex items-center gap-2">
  <StatusDot status="active" pulse />
  <span className="text-body-sm">Online</span>
</div>`}
