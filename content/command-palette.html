Command Palette

A keyboard-driven command palette for quickly navigating and performing actions within the application. Powered by the cmdk library.

The Command Palette uses the cmdk library (unstyled, accessible command menu). It provides CommandDialog for a modal overlay, CommandInput for search, CommandList for results, CommandEmpty for no-results state, CommandGroup for categorizing items, and CommandItem for individual entries. Keyboard shortcuts like Ctrl+K or Cmd+K toggle the dialog.

API

Component Description
Command Root container for the command menu.
CommandDialog Modal dialog wrapper that opens on keyboard shortcut (Ctrl+K / Cmd+K).
CommandInput Search input field for filtering commands.
CommandList Scrollable container for command groups and items.
CommandEmpty Displayed when no results match the search query.
CommandGroup Groups related commands under a heading.
CommandItem Individual selectable command entry with icon, label, description, and keyboard shortcut.

Usage

{`import {
  Command,
  CommandDialog,
  CommandInput,
  CommandList,
  CommandEmpty,
  CommandGroup,
  CommandItem,
  CommandShortcut,
} from '@/components/ui/command'

// Dialog mode (keyboard shortcut)
<CommandDialog open={open} onOpenChange={setOpen}>
  <CommandInput placeholder="Type a command or search..." />
  <CommandList>
    <CommandEmpty>No results found.</CommandEmpty>
    <CommandGroup heading="Navigation">
      <CommandItem onSelect={() => navigate('/tickets')}>
        <FileIcon />
        <span>Tickets</span>
        <CommandShortcut>G T</CommandShortcut>
      </CommandItem>
    </CommandGroup>
    <CommandGroup heading="Actions">
      <CommandItem onSelect={() => navigate('/tickets/new')}>
        <PlusIcon />
        <span>New Ticket</span>
        <CommandShortcut>C</CommandShortcut>
      </CommandItem>
    </CommandGroup>
  </CommandList>
</CommandDialog>

// Trigger with Ctrl+K
useEffect(() => {
  const down = (e: KeyboardEvent) => {
    if (e.key === 'k' && (e.metaKey || e.ctrlKey)) {
      e.preventDefault()
      setOpen((open) => !open)
    }
  }
  document.addEventListener('keydown', down)
  return () => document.removeEventListener('keydown', down)
}, [])`}
