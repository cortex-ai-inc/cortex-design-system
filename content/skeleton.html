Skeleton

A loading placeholder that mimics content layout using a pulsing animation. Used to indicate that content is being loaded.

React

Examples

Text Lines

Avatar + Text

Card Skeleton

Table Row

The Skeleton uses animate-pulse for a subtle opacity transition, bg-surface-container-high/50 for the background color, and rounded-sm for corners. Use it as a placeholder for any content that is still loading — text, avatars, cards, images, and table rows.

API

Prop Type Default Description
className string "" Additional CSS classes for width, height, or shape.

Usage

{`import { Skeleton } from '@/components/ui/skeleton'

// Text line
<Skeleton className="h-4 w-[250px]" />

// Avatar
<Skeleton className="h-10 w-10 rounded-full" />

// Card
<div className="flex flex-col gap-3">
  <Skeleton className="h-[120px] w-full rounded-lg" />
  <Skeleton className="h-4 w-[70%]" />
  <Skeleton className="h-4 w-[50%]" />
</div>`}
