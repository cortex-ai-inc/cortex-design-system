Breadcrumb

A navigation breadcrumb trail showing the current page location within the application hierarchy. Uses chevron separators and supports ellipsis for long paths.

React

Examples

Home Tickets TKT-0042
Dashboard Admin Team Members
Home Settings Profile

Breadcrumbs use chevron (ChevronRight) icons as separators. The current page is rendered as BreadcrumbPage (non-interactive <span>) while ancestors are BreadcrumbLink (interactive <a>). Use BreadcrumbEllipsis to truncate deep paths. A routeNames map can be used to display human-readable route names.

API

Component Props Description
BreadcrumbList className?: string Flex container <ol> for breadcrumb items.
BreadcrumbItem className?: string Individual breadcrumb item wrapper (<li>).
BreadcrumbLink href: string, asChild?: boolean Clickable breadcrumb link.
BreadcrumbPage className?: string Current page (non-interactive, bold text).
BreadcrumbSeparator className?: string Chevron icon separator between items.
BreadcrumbEllipsis className?: string Ellipsis icon for truncated paths.

Usage

{`import {
  Breadcrumb,
  BreadcrumbList,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbPage,
  BreadcrumbSeparator,
  BreadcrumbEllipsis,
} from '@/components/ui/breadcrumb'

const routeNames: Record = {
  dashboard: 'Dashboard',
  tickets: 'Tickets',
  admin: 'Admin',
}

<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink href="/home">Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbLink href="/tickets">Tickets</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>TKT-0042</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>`}
