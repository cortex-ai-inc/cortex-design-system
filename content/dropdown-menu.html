DropdownMenu

Contextual menu triggered by a button, providing a list of actions or options.

React Radix DropdownMenu

Overview

Dropdown menus display a list of actions or options anchored to a trigger element. Built on Radix DropdownMenu, they provide keyboard navigation (arrow keys, Enter, Escape), proper positioning with collision detection, and full ARIA menu pattern compliance.

The menu content uses bg-surface-container-low with ghost-border, rounded-sm, and shadow-ambient for elevation. Items are h-8 with horizontal padding and a hover state of bg-surface-container-high.

Structure

Component Role
DropdownMenu Root context provider
DropdownMenuTrigger Button/element that opens the menu (usually btn-ghost btn-icon)
DropdownMenuContent The popover panel containing menu items
DropdownMenuItem Individual action item (h-8, hover highlight)
DropdownMenuSeparator Visual divider between item groups
DropdownMenuLabel Section header label (uppercase, 11px, muted)
DropdownMenuGroup Groups related items together

Usage Guidelines

Do

  • Use destructive variant for irreversible actions
  • Group related items with separators and labels
  • Use icon + text for items to improve scannability
  • Provide keyboard shortcuts as secondary text

Don't

  • Overcrowd — limit to 7-10 items maximum
  • Use for navigation — prefer sidebar or topbar
  • Hide critical actions in dropdowns
