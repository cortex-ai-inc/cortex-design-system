Pagination

A pagination component for navigating through pages of content. Supports previous/next navigation and individual page links.

React

Example

...

Variants

Default
With Page Numbers Only

Pagination uses a consistent button-based layout. The current page is highlighted with the gradient-primary background. Previous/Next buttons include chevron icons and can show labels. Use the PaginationEllipsis component to indicate truncated page ranges.

API

Component Props Description
Pagination className?: string Root nav wrapper.
PaginationContent className?: string Flex container for pagination items.
PaginationItem className?: string Individual pagination item wrapper.
PaginationLink href?: string, isActive?: boolean Clickable page link. Active state applied when isActive is true.

Usage

{`import {
  Pagination,
  PaginationContent,
  PaginationLink,
  PaginationItem,
  PaginationPrevious,
  PaginationNext,
  PaginationEllipsis,
} from '@/components/ui/pagination'

<Pagination>
  <PaginationContent>
    <PaginationItem>
      <PaginationPrevious href="#" />
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="#" isActive>1</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="#">2</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="#">3</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationEllipsis />
    </PaginationItem>
    <PaginationItem>
      <PaginationNext href="#" />
    </PaginationItem>
  </PaginationContent>
</Pagination>`}
