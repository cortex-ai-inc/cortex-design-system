Icon Library

The Cortex design system uses lucide-react v0.562 as its sole icon library. All interface icons should be sourced from Lucide. Do not use inline SVGs, emoji, or images as interface icons — use Lucide components.

Note: The SVG paths shown below are for documentation purposes only. In production React code, import icons from lucide-react and use them as components: {">"}import { Bell } from "lucide-react"

Usage Guidelines

Do: Use icons as decorative elements alongside text labels to enhance scanability. Icons should support, not replace, text.
Do: Use aria-label when an icon is the sole indicator of an action or information (standalone icon buttons, status indicators).
Don't: Mix icon libraries. Stick to lucide-react exclusively.
Don't: Use emoji as interface icons. Reserve emoji for rich text content only.
Don't: Change strokeWidth from the default of 1.5. Consistency in line weight is critical for visual harmony.
