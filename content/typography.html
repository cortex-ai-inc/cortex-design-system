Default body text

text-body-sm (13px) is the default body size used throughout all Cortex products. It provides a compact, information-dense reading experience suitable for developer tools.

Usage Guidelines

✓ Do

Use text-body-sm (13px) as the default body text. Use text-label-sm (11px uppercase) for labels, section headers, and table headers. Use text-display-sm (36px) sparingly — only for primary page titles.

✗ Don't

Use font sizes outside the defined scale. Use more than one text-display-sm per page.
