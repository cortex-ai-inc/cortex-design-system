InputGroup

An input component with attached addons (prefix/suffix text or buttons). Useful for search fields, URLs, currency inputs, and more.

React

Addon Positions

Inline Start (Prefix)
$
Inline End (Suffix)
With Button Addon
Both Sides
https://

Button Addon Sizes

InputGroup composes an input with addon elements positioned at inline-start, inline-end, block-start, or block-end. Addons can be text labels, icons, or buttons. Button addons support xs, sm, icon-xs, and icon-sm sizes.

API

Component Props Description
InputGroup className?: string Root wrapper with flex layout and border.
InputGroupAddon align: "inline-start" | "inline-end" | "block-start" | "block-end", children: ReactNode Addon element attached to the input. Position controlled by align.

Usage

{`import { InputGroup, InputGroupAddon } from '@/components/ui/input-group'

// With prefix text
<InputGroup>
  <InputGroupAddon align="inline-start">
    <span>$</span>
  </InputGroupAddon>
  <input type="text" placeholder="Amount" />
</InputGroup>

// With suffix button
<InputGroup>
  <input type="text" placeholder="Search..." />
  <InputGroupAddon align="inline-end">
    <Button size="sm">Go</Button>
  </InputGroupAddon>
</InputGroup>`}
