Installation

Set up the Cortex Design System in your project.

Prerequisites

DependencyVersion
React19+
TypeScript5.9+
Tailwind CSS3.4+

Dependencies

Install the following packages:

npm install react@19 react-dom@19 typescript@5.9
npm install tailwindcss@3 @tailwindcss/typography tailwindcss-animate
npm install class-variance-authority clsx tailwind-merge lucide-react
npm install @radix-ui/react-avatar @radix-ui/react-dialog @radix-ui/react-dropdown-menu
npm install @radix-ui/react-select @radix-ui/react-separator @radix-ui/react-switch
npm install @radix-ui/react-tabs @radix-ui/react-tooltip @radix-ui/react-scroll-area
npm install @radix-ui/react-checkbox @radix-ui/react-slot
npm install sonner date-fns@4
npm install react-hook-form @hookform/resolvers zod
Tip

Install only the Radix UI primitives you actually need. The list above covers the full set used across Cortex products.

Full shadcn HSL layer

The complete :root block also carries the shadcn semantic variables (--background, --border, --primary, ...). Copy it verbatim from llm/getting-started.md — without it, shadcn components silently render unstyled.

Font Setup

Include Inter and JetBrains Mono in your HTML:

<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=JetBrains+Mono:wght@400;500&display=swap"
  rel="stylesheet"
/>

Then apply them via CSS:

body {
  font-family: 'Inter', system-ui, sans-serif;
  font-size: 13px;
  color: var(--on-surface);
  background-color: var(--surface-dim);
}

code, pre {
  font-family: 'JetBrains Mono', monospace;
  font-size: 12px;
}

cn() Utility

The cn() utility merges Tailwind classes, resolving conflicts intelligently:

import { clsx, type ClassValue } from 'clsx';
import { twMerge } from 'tailwind-merge';

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}

shadcn/ui Setup

Configure components.json:

{
  "$schema": "https://ui.shadcn.com/schema.json",
  "style": "new-york",
  "rsc": true,
  "tsx": true,
  "tailwind": {
    "config": "tailwind.config.js",
    "css": "src/index.css",
    "baseColor": "slate",
    "cssVariables": true
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils",
    "ui": "@/components/ui"
  },
  "iconLibrary": "lucide"
}
