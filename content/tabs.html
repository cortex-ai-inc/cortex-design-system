Tabs

Tabbed navigation for switching between related content sections.

React Radix Tabs

Overview

Tabs organize content into separate views accessible via a tab bar. Built on Radix Tabs, they provide full keyboard navigation (arrow keys switch tabs, Tab moves into content), proper ARIA tab pattern, and automatic content visibility management.

The tab list (TabsList) uses bg-surface-container-high (#282A30) with rounded-sm (4px) and p-1 padding. Each trigger (TabsTrigger) is inactive by default and activates with bg-surface-container-low (#191B22) fill.

Usage Guidelines

Do

  • Keep tab labels short — one to two words
  • Use badges to indicate counts on tabs
  • Ensure content in each tab is independently meaningful

Don't

  • Use tabs for navigation between unrelated pages — use sidebar or topbar
  • Put more than 6-7 tabs in a single list
  • Use vertical tabs in narrow containers
