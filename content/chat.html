Chat Messages

Chat messages use an Item component that renders a message bubble with optional sender information, timestamp, and actions. Messages alternate between user (right-aligned) and assistant (left-aligned), though in support contexts all messages are typically left-aligned with the agent's name and avatar.

Message Actions

Each message can have contextual actions that appear on hover or are always visible for the last message: copy, edit, delete, retry (for failed messages).
