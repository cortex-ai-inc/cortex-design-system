Checkbox

Binary selection control for toggling an option on or off.

React Radix Checkbox

Overview

The Checkbox component uses Radix Checkbox for accessible, keyboard-navigable binary selection. Checkboxes are w-4 h-4 with rounded-sm (4px) corners. When checked, the background fills with bg-k-primary-container (#4D8EFF) and displays a white checkmark icon. The unchecked state shows a ghost-border-20 border with no fill.

Checkboxes should always be paired with a Label component for accessibility. The label uses text-body-sm (13px) and sits to the right of the checkbox with gap-2 spacing.

Usage Guidelines

Do

  • Always pair with a visible label
  • Use for multi-select lists of options
  • Use for binary settings (on/off)

Don't

  • Use for mutually exclusive options — use RadioGroup instead
  • Trigger actions immediately on check — pair with a submit button
