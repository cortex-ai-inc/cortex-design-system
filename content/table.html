Table

A responsive data table component for displaying tabular data with consistent styling.

Example

List of recent support tickets
Ticket Subject Status Priority Created
TKT-0042 Cannot connect to VPN after update In Progress High 2h ago
TKT-0041 New feature request: dark mode toggle Open Medium 1d ago
TKT-0040 Billing invoice for June 2026 Resolved Low 3d ago

The Table component wraps native HTML tables with consistent styling. It is responsive by wrapping in a scrollable container with a min-width breakpoint. Rows have a hover state using hover:bg-muted/50 and dividers using the ghost-border utility.

API

Prop Type Default Description
className string -- Additional CSS classes
children ReactNode -- Table content (thead, tbody, caption)

Usage

{`import { Table, TableHeader, TableBody, TableRow, TableHead, TableCell, TableCaption } from '@/components/ui/table'

export function TicketTable() {
  return (
    <Table>
      <TableCaption>Recent support tickets</TableCaption>
      <TableHeader>
        <TableRow>
          <TableHead>Ticket</TableHead>
          <TableHead>Subject</TableHead>
          <TableHead>Status</TableHead>
        </TableRow>
      </TableHeader>
      <TableBody>
        <TableRow>
          <TableCell className="font-mono">TKT-0042</TableCell>
          <TableCell>Cannot connect to VPN</TableCell>
          <TableCell><Badge variant="warning">In Progress</Badge></TableCell>
        </TableRow>
      </TableBody>
    </Table>
  )
}`
