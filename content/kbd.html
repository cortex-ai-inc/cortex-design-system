Kbd

Keyboard key chip for documenting shortcuts. Native <kbd> element, no library.

New React

Overview

Kbd renders a single physical key as a mono 11px chip — h-5, rounded-sm, bg-surface-container-high with a ghost border and text-on-surface-variant. KbdGroup lines up multiple keys for combinations, with an optional + separator at text-on-surface-variant/50.

Non-interactive like Badge: no hover, focus, or active states — Kbd displays a key, it never triggers the shortcut.

Usage Guidelines

Do

  • Use only for physical keyboard keys and combos
  • Use standard glyphs: ⌘ ⇧ ⌥ Ctrl Alt ↵ Esc
  • Use KbdGroup with the muted + separator for combos

Don't

  • Make Kbd clickable — use a Button if the shortcut should be triggerable
  • Use for arbitrary emphasis or tags — use Badge or code instead
  • Restyle per context — the chip is fixed at mono 11px, h-5
