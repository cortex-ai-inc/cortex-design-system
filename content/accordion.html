Accordion

A vertically stacked set of interactive headings that expand and collapse content sections. Powered by Radix UI.

React Radix

Example

A support ticket is a record of a customer issue or request that needs to be addressed by the support team. Each ticket contains details about the problem, its priority, status, and communication history.
Navigate to Tickets > New Ticket and fill in the required fields including subject, description, and priority. Our team will respond as soon as possible.
Tickets can be marked as Low, Medium, High, or Critical. High and Critical priority tickets are escalated to senior team members for faster resolution.

The Accordion uses Radix UI's Accordion.Root, Accordion.Item, Accordion.Trigger, and Accordion.Content. Items are separated by border-b. The chevron icon rotates on open/collapse using a CSS transition. Supports both single and multiple open items via the type prop.

API

Prop Type Default Description
type "single" | "multiple" "single" Accordion mode. "single" allows one open item, "multiple" allows several.
collapsible boolean true Allow all items to be closed simultaneously (single mode).
defaultValue string | string[] -- Initially open item value(s).

Usage

{`import {
  Accordion,
  AccordionContent,
  AccordionItem,
  AccordionTrigger,
} from '@/components/ui/accordion'

<Accordion type="single" collapsible defaultValue="item-1">
  <AccordionItem value="item-1">
    <AccordionTrigger>What is a support ticket?</AccordionTrigger>
    <AccordionContent>
      A support ticket is a record of a customer issue or request.
    </AccordionContent>
  </AccordionItem>
  <AccordionItem value="item-2">
    <AccordionTrigger>How do I create a ticket?</AccordionTrigger>
    <AccordionContent>
      Navigate to Tickets &gt; New Ticket and fill in the required fields.
    </AccordionContent>
  </AccordionItem>
</Accordion>`}
