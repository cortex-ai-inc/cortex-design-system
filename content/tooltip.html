Tooltip

A popup that displays information related to an element when the element receives keyboard focus or the mouse hovers over it, powered by Radix UI.

React Radix

Example

What is this?
Additional context about this element

Tooltip content renders with bg-surface-container-high (#282A30) background, shadow-ambient, and text-code-sm (12px) typography. The delayDuration prop controls the open delay (default 700ms).

API

Component Props Description
TooltipProvider delayDuration?: number Wraps the app, provides tooltip configuration. Default delay: 700ms.
Tooltip open?: boolean, defaultOpen?: boolean Root container for trigger and content.
TooltipTrigger asChild?: boolean The element that triggers the tooltip.
TooltipContent side?: 'top' | 'right' | 'bottom' | 'left', sideOffset?: number The tooltip popup content. Default side: top, offset: 4.

Usage

{`import {
  Tooltip,
  TooltipContent,
  TooltipProvider,
  TooltipTrigger,
} from '@/components/ui/tooltip'

<TooltipProvider delayDuration={300}>
  <Tooltip>
    <TooltipTrigger asChild>
      <Button variant="ghost">Hover</Button>
    </TooltipTrigger>
    <TooltipContent side="top">
      <p>This is a tooltip</p>
    </TooltipContent>
  </Tooltip>
</TooltipProvider>`}
