Cortex Design System

Build consistent, accessible, dark-themed interfaces for Cortex AI products.

React 19 Radix UI TypeScript

Design Principles

Dark by default

A dark-first system designed for developer tools that are used for extended sessions. Reduces eye strain and emphasizes content.

Tonal layering

Surface colors create depth without drop shadows. Each layer sits higher than the last — dim, lowest, low, high — forming a natural hierarchy.

Blue gradient accent

The signature Cortex blue gradient — from #ADC6FF to #4D8EFF — is the primary visual identifier across all products.

Compact & efficient

Default body text at 13px (text-body-sm) keeps interfaces dense and information-rich. Every pixel is deliberate.

Semantic color

Colors carry meaning: green for success, red for errors, yellow for warnings. Used consistently across all products for immediate recognition.

Accessibility-first

ARIA labels, keyboard navigation, focus indicators, and contrast ratios are integral — not afterthoughts. Radix UI primitives provide a solid foundation.

Developer experience

Type-safe with TypeScript 5.9, predictable class naming, minimal abstraction. The cn() utility and CVA patterns make styling straightforward.
