RadioGroup

Mutually exclusive selection control — pick exactly one option from a small visible set.

New React Radix RadioGroup

Overview

RadioGroup uses Radix RadioGroup for accessible single-choice selection with arrow-key navigation. Items are 16x16 circles (rounded-full — sanctioned here because the control is inherently circular). The checked state mirrors Checkbox exactly: the circle fills with bg-k-primary-container (#4D8EFF) and shows a dark 8px dot (fill-surface-container-lowest) — a deliberate divergence from stock shadcn's outline-plus-blue-dot.

The unchecked state is a border-on-surface-variant/30 outline with no fill. Always pair each item with a Label (text-body-sm, gap-2). The Root lays out as grid gap-2. Single 16px size — no size prop.

Usage Guidelines

Do

  • Use for 2–5 mutually exclusive options, all visible at once
  • Always pair each item with a visible label via htmlFor
  • Use items-start + mt-0.5 when options have descriptions

Don't

  • Use for more than 5 options — use Select instead
  • Use for independent toggles — use Checkbox instead
  • Revert to shadcn's outline + blue dot — checked is filled blue with a dark dot
