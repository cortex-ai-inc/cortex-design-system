Animations

Subtle, purposeful animations that enhance without distracting.

Motion Scale

Three named durations, three named easings. Everything else is a component constant.

TokenValueRole
--motion-fast150msMicro-interactions: hover fills, color/border/opacity
--motion-base200msEntrances/exits: fade-in, slide-up, accordion, dropdown
--motion-slow300msLarge surfaces: Sheet/Drawer panels, layout shifts
--ease-outcubic-bezier(0, 0, 0.2, 1)Default — entrances, hovers
--ease-in-outcubic-bezier(0.4, 0, 0.2, 1)On-screen movement, resizes
--ease-linearlinearInfinite loops only (spinner, pulse-ring)

Animation Principles

  • 200ms default — fast enough to feel responsive, slow enough to perceive
  • Nothing over 300ms — except infinite loops (spinner, pulse, skeleton)
  • ease-out — decelerate naturally; no bounce or spring effects, ever
  • Subtle shifts — 10px translateY for slide-up, opacity only for fade-in
  • Infinite animations — reserved for loading/status indicators only
  • Respect prefers-reduced-motion: reduce — disable non-essential animation
