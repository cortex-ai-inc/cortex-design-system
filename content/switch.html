Switch

Toggle control for binary settings with immediate effect.

React Radix Switch

Overview

The Switch component uses Radix Switch for accessible, keyboard-navigable binary toggling. Switches are used for settings that take effect immediately, without requiring a form submission. The track is h-5 w-9 (20px x 36px) with rounded-full, and the thumb is h-4 w-4 (16px) that translates translate-x-4 when toggled on.

The unchecked track uses bg-surface-container-high (#282A30). When checked, it fills with bg-k-primary-container (#4D8EFF). The thumb is always white. Switches should always be paired with a Label for accessibility.

Usage Guidelines

Do

  • Use for settings with immediate effect (no submit needed)
  • Always pair with a descriptive label
  • Use Switch over Checkbox for settings panels

Don't

  • Use Switch for multi-select options — use Checkbox instead
  • Use Switch for actions that require confirmation
  • Use without a label — the binary state needs context
