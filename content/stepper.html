Stepper

Shows progress through an ordered multi-step flow with 24px circular indicators and connector lines. Heads wizard pages — steps are pages or page sections, never modals.

New React

Overview

A custom compound component — no Radix primitive. The React implementation renders <nav aria-label="Progress"> wrapping an <ol> with one <li> per step; the current item carries aria-current="step". Indicator colors follow the StatusDot semantics: blue fill with a dark glyph (bg-k-primary-container + text-surface-container-lowest) for current and completed steps, red tints for errors, ghost outline for pending. The circular indicator is a sanctioned rounded-full use. See stepper.md for the full spec.

Error state

Mark a step error when its validation fails so users can see where to return. The indicator uses the semantic tint ladder (bg-k-error/20 fill, border-k-error/40), and the title turns red.

Wizard pattern

The stepper sits at the top of a wizard page; the current step's form renders below it (standard form grid, max-w-5xl). The footer pairs a secondary "Back" on the left with the primary gradient CTA on the right — on the final step the CTA becomes the submit action. Validate the current step before advancing and keep entered values when navigating Back. See the Forms pattern for grid and validation rules.

Prop Type Default Description
value number — This step's index (1-based)

StepperTrigger wraps the indicator in a <button> so users can jump back to completed steps — pending steps stay disabled. Trigger focus ring: focus-visible:ring-2 focus-visible:ring-ring focus-visible:ring-offset-2.

Usage Guidelines

Do

  • Keep wizards to 3–5 steps
  • Mark failed steps with the error state
  • Keep titles to 1–2 words; put detail in descriptions
  • Preserve entered values when navigating Back

Don't

  • Exceed 6 steps — split the flow instead
  • Build modal wizards — steps are pages or page sections
  • Skip validation or allow jumping ahead to pending steps
  • Put light text on the blue indicator fill — the glyph is dark
