Progress

A progress bar component powered by Radix UI for indicating progress or loading states. Supports determinate and indeterminate modes.

React Radix

Examples

Loading... 45%
Upload progress 72%
Completed 100%
Indeterminate

The Progress component uses a h-2 rounded-full bg-surface-container-high track with a bg-k-primary-container indicator that transitions smoothly via transition-all. The indeterminate state shows an animated bar using a sliding CSS animation.

API

Prop Type Default Description
value number 0 Progress value (0-100). Omit for indeterminate state.
className string "" Additional CSS classes.

Usage

{`import { Progress } from '@/components/ui/progress'

// Determinate
<Progress value={45} />

// Indeterminate
<Progress />

// With label
<div className="flex flex-col gap-2">
  <div className="flex justify-between">
    <span className="text-body-sm">Uploading...</span>
    <span className="text-body-sm text-muted">45%</span>
  </div>
  <Progress value={45} />
</div>`}
